# VSCode cool keyboard shortcuts
```
CMD + ยง - Close active editor
CMD + . - Open Quick fix
CMD + E - Search everywhere/commands
CMD + W - Expand selection
CMD + 1 - Focus left bar
CMD + 2 - Focus first editor
CMD + 3 - Focus second editor (splits screen)
```
